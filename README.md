# Lathmar GitHub Pages Website Draft

Simple static website draft for GitHub Pages.

## Files

- `index.html`
- `style.css`
- `assets/main-screen.jpg` placeholder path

## Setup

1. Copy `index.html` and `style.css` into your public GitHub Pages repository.
2. Create an `assets` folder.
3. Export your Lathmar main screen/title image as:

   `assets/main-screen.jpg`

4. Replace the YouTube placeholder in `index.html`:

   `VIDEO_ID`

   with your real YouTube video ID.

5. Optionally replace the YouTube button URL with your actual channel URL.

## GitHub Pages

Use:

Settings → Pages → Deploy from a branch → main → /root
