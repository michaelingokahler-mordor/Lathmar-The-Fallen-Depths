About the Game

Lathmar: The Fallen Depths is a modern dungeon crawler built around the spirit of the classic 90s RPGs: party creation, guilds, dangerous expeditions, monster encounters, loot, and the slow descent into a hostile underworld.

The game aims to keep the old-school feeling intact while improving the structure around it: clearer character management, a stronger guild system, a modern interface, and systems inspired by tabletop role-playing mechanics.

Development is currently in alpha. The focus is on building the core game loop, dungeon exploration, character progression, item systems, and the tools needed to create larger dungeon scenarios.

  • Classic party-based dungeon crawling
  • Guilds, races, stats, equipment, and progression
  • Dark fantasy presentation with retro RPG roots
  • Built as a tribute to the games that defined the genre
